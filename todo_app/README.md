->Smart Todo App:
A simple and interactive Todo application built using HTML, CSS, and JavaScript.

->Features:
Add tasks
Delete tasks
Mark tasks as complete
Edit tasks
Filter (All / Completed / Pending)
Search tasks
Data saved using localStorage

->Files:
index.html → Structure
style.css → Styling
script.js → Functionality

->How to Use
Enter a task
Click Add
Manage tasks using buttons
Use filters and search

->Note
Tasks are saved in browser using localStorage, so they remain after refresh.
